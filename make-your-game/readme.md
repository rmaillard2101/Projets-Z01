Commandes du jeu

Ce document décrit les commandes utilisées pour contrôler le jeu Tetris.


Déplacements

Flèche gauche (←) : Déplacer la pièce actuelle vers la gauche.

Flèche droite (→) : Déplacer la pièce actuelle vers la droite.

Flèche bas (↓) : Accélérer la chute de la pièce actuelle.


Rotation

Flèche haut (↑) : Faire pivoter la pièce actuelle (sauf la pièce carrée O).


Autres commandes

Espace ( ) : Démarrer le jeu ou relancer après une pause.

Échap (Esc) : Mettre le jeu en pause ou reprendre après pause.


Interface

Bouton "Play" : Démarrer le jeu.

Bouton "Pause" : Mettre le jeu en pause.

Bouton "Restart" dans le menu pause : Redémarrer le jeu depuis le menu pause.

Bouton "Resume" dans le menu pause : Reprendre le jeu depuis le menu pause.
